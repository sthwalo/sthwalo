# FIN Marketing Videos

This directory contains the demo videos for the FIN product walkthrough.

## File Structure

```
videos/
├── fin-demo-01-signup.mp4
├── fin-demo-02-company-setup.mp4
├── fin-demo-03-dashboard.mp4
├── fin-demo-04-transactions.mp4
├── fin-demo-05-reports.mp4
├── fin-demo-06-payroll.mp4
├── fin-demo-07-audit-trail.mp4
└── fin-demo-08-multi-company.mp4
```

## Video Specifications

- **Format**: MP4 (H.264)
- **Duration**: 30 seconds each
- **Resolution**: 1920x1080 (Full HD)
- **Frame Rate**: 30fps
- **Bitrate**: 5-8 Mbps

## Content Guidelines

Each video should demonstrate one step of the FIN workflow:

1. **Signup**: User registration and trial activation
2. **Company Setup**: Creating company profile and workspace
3. **Dashboard**: Financial dashboard overview
4. **Transactions**: Transaction processing and classification
5. **Reports**: Financial report generation
6. **Payroll**: Payroll management
7. **Audit Trail**: Compliance and audit visibility
8. **Multi-Company**: Managing multiple businesses

## Thumbnails

Corresponding thumbnails should be placed in `/public/images/demo-thumbnails/` with matching names (e.g., `signup.jpg`, `dashboard.jpg`).
